# Приоритетный шифратор
Реализация приоритетного шифратора с выходным признаком четности и изолированности вектора. Асимптотика критического пути данного модуля определяется двумя строчками:

```
assign IsolatedTopBit[i] = vector[i] & ~(|(vector[WIDTH - 1: i + 1]));
                            ...
assign position[i] = |(IsolatedTopBit & mask);

```

сначала происходит log(WIDTH) побитовых ИЛИ (если будет реализовано иерархическим способом) и НЕ с WIDTH - 1 битами вектора. Потом с получившимся новым вектором проводится побитовое И и еще log(WIDTH) побитовых ИЛИ с WIDTH битами. Получившаяся асимптотика: O(log(WIDTH)).
